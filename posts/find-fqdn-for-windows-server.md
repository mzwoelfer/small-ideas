# [[Find FQDN for Windows Server]]
**Where to Find the FQDN?**

1.  Launch the **Control Panel** by searching for “Control Panel” in the Start Menu, or by typing Win+R and typing “control.exe” in the Run menu.
2.  Click on the “**System & Security**” menu in the Control Panel.
3. Click "**System**"
4.  On the System Information screen, you will see both the **hostname** and **[[FQDN]]** of your machine.
